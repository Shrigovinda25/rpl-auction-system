# 🏆 RPL Auction System 2026

A high-performance, real-time live auction bidding system for player leagues. Features an interactive Admin Panel for bidding and a Projector Screen for live display.

## 🚀 One-Click Deployment to Vercel

1. **Upload to GitHub**:
   - Create a new repository on GitHub.
   - Run the following in your terminal:
     ```bash
     git init
     git add .
     git commit -m "Initial commit: RPL Auction System Ready"
     git branch -M main
     git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
     git push -u origin main
     ```
2. **Connect to Vercel**:
   - Go to [vercel.com](https://vercel.com).
   - Click **"Add New Project"**.
   - Select your GitHub repository.
   - Click **"Deploy"**.

## 🛠️ Features
- **Real-Time Bidding**: Cross-tab synchronization via `localStorage`.
- **Excel Ingestion**: Support for complex headers, multi-sport players, and Google Drive images.
- **Projector Optimized**: Slanted "Selected" stamps and high-impact visual feedback.

## 📁 Project Structure
- `index.html`: Entry portal.
- `admin.html`: Bidding and management control center.
- `auction.html`: Live projection screen.
- `js/`: Core logic and synchronization.
